# M.E.S.A. | Secretário Virtual v4.5 🏛️🤖

> **Status:** Estado da Arte - Jornada de Automação Estratégica
> **Autor:** Robert Macedo Lima (Designer de Ambientes / UEMG)

O **M.E.S.A.** (Mentoria Estratégica Sistêmica Autônoma) é um ecossistema de atendimento inteligente desenvolvido para otimizar a gestão de aulas particulares de AutoCAD e Revit. 

### 🛠️ Tecnologias de Alta Performance
- **Cérebro Cloud:** GLM-4.7 via Ollama Cloud API.
- **Conexão Web:** Busca em tempo real via DuckDuckGo Search API.
- **Interface de Comunicação:** Telegram Bot API (Assíncrono).
- **Notificação de Lead:** Integração com Desktop (Windows Bips) e Celular.

### 🌿 Estética & Filosofia
O projeto segue a linha **Industrial Organic Sophisticated**, unindo a frieza da automação de alta performance com a fluidez do atendimento humanizado e estratégico.
